¿Cómo usar los iconos?

Usando SVG con el componente:

import IconoAnimal from '../components/ui/IconoAnimal';

// Luego, en tu JSX:
<IconoAnimal tipo="toro" tamaño={32} />
<IconoAnimal tipo="vaca" />
<IconoAnimal tipo="vaca_amamantando" />

Usando PNG optimizados:

import OptimizedImage from '../components/ui/OptimizedImage';

// Luego, en tu JSX:
<OptimizedImage 
  src="/images/toro.png" 
  alt="Toro" 
  width={32} 
  height={32} 
/>
